# PR Title

docs: add architecture review report

## Summary

Adds a repository-level architecture/code review report for CarrotGuard.

The report focuses on delivery readiness, Unity runtime integration risks, test coverage gaps, and maintainability issues found during static review.

## Key Findings Covered

- Fresh clone is not a reproducible runnable Unity project because scene, prefab, `ProjectSettings`, `Packages`, and serialized assets are missing.
- HUD subscription can be missed depending on Unity execution order.
- Enemy path setup lacks null/length validation and can crash at runtime.
- Tower targeting uses per-frame global lookup and allocations.
- Economy updates leak `GoldSystem` through UI code and use `AwardGold(0)` as a refresh hack.
- Existing tests cover pure logic but not Unity lifecycle, prefab wiring, UI interaction, or scene-level behavior.
